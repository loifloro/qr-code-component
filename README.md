### What I learned

I learned in this activity how to properly center a div using grid and place-items: center. 

```css
body {
    background: $light-gray;
    font-family: Outfit, sans-serif;
    height: 100vh;
    display: grid;
    place-items: center;
}
.card {
    background: $white;
    max-width: 330px;
    border-radius: 1rem;
    padding: 1rem;
    margin: auto;
}
```
